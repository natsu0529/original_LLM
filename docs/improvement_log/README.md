# Improvement Log

This directory tracks the autonomous improvement attempts on the friend-style Dazai chat LLM.

Format for each session: a dated markdown file under this directory.
Each entry includes:

- 観察された問題 (observed problem)
- 推定原因 (hypothesized cause)
- 対処 (action taken)
- 結果 (resulting behavior)
- 次の一手 (next step)

This is intentionally append-only. Old entries are kept even if later disproven, so we can trace how reasoning evolved.
